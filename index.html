vuejs-scroll-fixed-header v1.0.3

A simple vue 2 scroll fixed header component

About

This is a simple Vue2 component that achieve the purpose of scroll to certain height, the given header will fixed at the top of the browser window

Usage

        
          <scroll-fixed-header :fixed.sync="fixed" :threshold="56">
            <nav class="navbar navbar-light bg-light">
              <div class="container">
                <a class="navbar-brand" href="#">
                  Vuejs Scroll Fixed Header
                </a>
              </div>
            </nav>
          </scroll-fixed-header>

Props

name type default description
fixed Boolean false The fixed status of the current header
threshold Number 0 The scroll top threshold for determining the fixed status
